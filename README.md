# numberplace

Sudoku.javaを実行するとナンプレの盤面(9×9)が表示されます。
<br>盤面右のラジオボタンで問題の種類を選択します。各問題につい
<br>ては後述します。
<br>盤面上の各マスをクリックするとマスが灰色になり、選択された
<br>状態になります。選択されたマスに数字を入れるには盤面上部の
<br>１〜９の数字ボタンを使います。マスを空白にするときは一番左
<br>のボタンを使います。このようにしてナンプレの盤面を入力しま
<br>す。盤面を入力したあとOKボタンを押すとそのナンプレを解いた
<br>結果が標準出力に出力されます。

## 問題の種類
<br>Basic   : 通常のナンプレです。
<br>Diagnoal: 対角線上のマスにも１〜９の数字が一つずつ入るよう
<br>　　　　　にするという条件が追加されたナンプレです。これを
<br>　　　　　選択すると対角線上のマスには斜めに線が引かれます。
<br>Even-Odd: 特定のマスには偶数のみ、あるいは奇数のみが入るよ
<br>　　　　　うにするという条件が追加されたナンプレです。奇数
<br>					か偶数かは各問題で異なります。これを選択すると盤
<br>　　　　　面上部のE-Oボタンが使えるようになります。このボタ
<br>　　　　　ンを押すと選択されたマスが緑色になります。このよ
<br>　　　　　ようにして偶数のみ、奇数のみが入るマスを指定しま
<br>　　　　　す。

## そのほか
<br>・Diagnoalを押した後Even-Oddを押すと対角線上に線が残ったままに
<br>　なりますが今のところ仕様です。
<br>・Zigzagは、通常のナンプレにある3×3の固まりを、任意の連続した
<br>　9マスにしたナンプレですがまだ実装していません。
<br>・答えが二つ以上ある場合はすべて表示されます。
